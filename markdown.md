# markdown基本语法 
## 基本符号：* - + . >  
基本命令都是由基本符号构成  
## 标题
1-6级标题时间1到6个# 代表  #号后面有一个空格
```markdown
# 1级标题
## 2级标题
...
###### 6级标题
```
## 无序列表
使用+ - * 标识列表项目，嵌套使用tab或者空格
```markdown
+ a
+ b
+ c
	+ c1
	+ c2
+ d
```
## 有序列表
使用数字
```markdown
1. a
2. b
3. c
```
## 引用说明快  
`> 引用内容`  
嵌套应用使用`>>嵌套`
## 代码块
使用 \` \`或者\``` \```
## 链接
`[链接文字](http://www.链接地址.com) title属性`  
`[链接文字]: http://www.链接地址.com "title属性"`
## 图片
`![图片文字](http://www.图片地址.com) title属性`  
`![图片文字]: http://www.图片地址.com "title属性"`
## 分割线 * - _ 组成，至少3个
`***  ----  _____`
## 强调字体 
`*斜体* **加粗** _斜体_  __加粗__  ~~删除线~~`
## 表格
```markdown
|123|123|13|
|:-|:-|:-|
|abc|abc|abc|
```
